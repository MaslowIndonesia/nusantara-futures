# Nusantara Futures — live research terminal

Auto-generated dashboard for an institutional futures research fund (6 global markets:
XAUUSD, US30, US100, US500, UK100, JP225). Dual-panel Monte-Carlo book (generic stress
floor vs the broker's full price history), daily long/short signals, profitability
tracking, and a forward paper-ledger.

**Research / paper only. No number here is a promise, a recommendation, or a live trade.**

`index.html` is a self-contained static render (no parameters, no source, no external
assets). It is regenerated by the fund's 5×/day loop and pushed here automatically.
